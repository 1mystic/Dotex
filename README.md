# Matex

Matex is a browser-based Markdown and LaTeX editor with live preview, export to PDF/HTML/Markdown, and static deployment support for Netlify and Vercel.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

The app is configured as a single-page application. Netlify and Vercel rewrites are included so direct route hits resolve correctly.
